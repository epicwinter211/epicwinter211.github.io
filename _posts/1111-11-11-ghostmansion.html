What is it?

Ghost mansion is my first minecraft minigame I have made. It is a multiplayer game. You play it by typing:

/start

If you are a ghost hunter, the objective is to shine your flashlight on the ghost. If you are the ghost, the objective is to scare all the players (or walk into them.)

You are timed as the ghost tries to get you. If the ghost doesnt get you in time, the players win. If the ghost scares all the players, the ghost wins.

To download it, click here.

The config file holds some variables. The config looks like this:

delayToStart: 20 delayNewGameStart: 20 autoStart: true serverName: §b§lCycle§aMC messageDelay: 300

The

delayToStart: 20
means how long the game will wait when it is counting down (in seconds)

The

delayNewGameStart: 20
means how long the game will take when it is autostarting the game. This variable won't matter if you set autoStart to false.

autoStart: true
sets if the game will automatically start or not once a game ends.

The variable

serverName: §b§lCycle§aMC
is the name of your server. It uses minecraft color codes.
