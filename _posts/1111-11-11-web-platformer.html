What is it?

The web platformer is a game that I am trying to create with pure js and a canvas.

The purpose of this project is to see how far I can get without using an engine like phaser.

I have made an engine that will take a tilemap and put it on the page, and you can collide with blocks.

It is a little buggy, where you sometimes can jump into a wall. The keys to move are WASD.

I looked at scratch, a game maker that I started learning to code in, and used a physics engine from that.

It surprisingly worked well!
